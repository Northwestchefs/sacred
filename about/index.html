About Sacred

A careful Hebrew Bible study project

Sacred is an early-stage project focused on building a calm, trustworthy environment for Hebrew Bible reading and study.

The current MVP prioritizes clear reading flow, preserved chapter and verse numbering, and static-site reliability before advanced features.

Text care

Hebrew text presentation is treated as primary, with right-to-left readability and minimal visual noise.

Sources and numbering

Sacred aims to preserve source structure and numbering so citations and study references remain stable across tools.

Sources and attribution

Sacred documents source provenance so readers can see where text data comes from and how it may be reused.

  • Hebrew source line (candidate): Open Scriptures Hebrew Bible / Westminster Leningrad Codex (WLC). We are still confirming the exact canonical upstream URL before production import. In the meantime, see Open Scriptures and Sacred's Hebrew source manifest.
  • English translation source: The Holy Scriptures: A New Translation (JPS 1917) by the Jewish Publication Society. Source page · Public-domain notice
  • API and translation credits: Sacred uses the Sephira API for text access and credits Sephira for any translation content retrieved through that API.
  • Licensing snapshot: JPS 1917 is treated as public domain. For Hebrew-source planning, Sacred currently treats the WLC text line as public domain and Open Scriptures annotation layers as CC BY 4.0 (both pending final upstream verification). See Sacred's Hebrew licensing notes and CC BY 4.0 license text.
