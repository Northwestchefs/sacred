# Reference Data

The `reference/` directory stores source documentation, licensing notes, schemas, and raw/processed reference data used by Sacred.

Its purpose is to preserve:

- **Provenance** (where source data came from and when it was accessed)
- **Legal clarity** (what may be copied, transformed, redistributed, and attributed)
- **Consistent data handling** (repeatable import and normalization conventions)

Use this directory as the canonical location for source-of-truth documentation before data is consumed by application code.
