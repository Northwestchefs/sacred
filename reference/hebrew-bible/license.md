# Hebrew Bible Licensing Notes

## Current Understanding
Based on currently available source documentation:

- The Westminster Leningrad Codex (WLC) **text** is understood to be in the **public domain**.
- Open Scriptures **lemma and morphology annotations** are understood to be licensed under **CC BY 4.0**, which requires attribution.

These points must be verified directly against the source project pages and/or repository license files before production use. Where exact terms are not yet finalized in this repository, treat them as **to be confirmed from source**.

## Verification Checklist
Before importing or redistributing data, confirm and document:

- [ ] Source text rights (public domain or other explicit rights statement)
- [ ] Annotation rights (lemma/morphology licensing terms)
- [ ] Required attribution language and placement
- [ ] Redistribution permissions (including packaged data releases)
- [ ] Modification permissions (including transformed/normalized outputs)
- [ ] Commercial use status

## Attribution Planning
If annotation layers are CC BY 4.0, ensure Sacred includes:

- Source name and URL
- License name and link
- Any specified attribution text from upstream
- A clear indication of modifications, if any

## Compatibility Warning
Do **not** merge third-party annotations into Sacred project data until license compatibility is explicitly confirmed and documented.
