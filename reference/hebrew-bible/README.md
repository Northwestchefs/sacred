# Hebrew Bible Reference

This directory is the canonical location for Hebrew Bible source documentation and import structure used by Sacred.

## Purpose

- Track source provenance and release details for Hebrew Bible text inputs.
- Record licensing and attribution requirements.
- Define the schema and transformation expectations for project data pipelines.

## Storage Layout

- `raw/`: source files kept as close as possible to their original/downloaded form.
- `processed/`: normalized files transformed for project use (stable structure, consistent encoding, and predictable field naming).

Keep source documentation updated before introducing new text or annotation assets.
